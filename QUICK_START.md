# 🚀 빠른 시작 가이드

프로젝트를 빠르게 시작하는 방법입니다.

## ⏱️ 5분 안에 시작하기

### 1단계: 프로젝트 구조 생성 (10초)
```bash
python setup_project.py
```

### 2단계: API 키 발급 (10분)
**[API_KEY_GUIDE.md](API_KEY_GUIDE.md)** 파일을 열어서 단계별로 따라하세요.

필요한 API 키:
- ✅ 서울시 일반 인증키 (즉시)
- ✅ 서울시 실시간 지하철 인증키 (즉시)
- ⏰ 기상청 단기예보 인증키 (1-2일)

### 3단계: .env 파일 설정 (1분)
```bash
# .env.template을 .env로 복사
copy .env.template .env  # Windows

# .env 파일을 열어서 API 키 입력
```

### 4단계: 라이브러리 설치 (2분)
```bash
pip install -r requirements.txt
```


### 5단계: 테스트 실행 (30초)
```bash
python src/data_collector.py
```
